## Iniciando o projeto

Primeiramente será necessário clonar este projeto em (https://github.com/BrunoCarvalhoFeitosa/layers-saas.git), após o download será necessário abrir este projeto no seu editor e no terminal digitar npm install, posteriormente é só rodar em seu terminal o comando npm run dev, após isso, a página será aberta em seu navegador. Lembrando que são necessárias algumas variáveis ambiente para que de fato a aplicação funcione sem erros.

### Pré-requisitos

* npm
  ```sh
  npm install npm@latest -g
  ```

### Instalação

1. Clone o repositório
   ```sh
   git clone https://github.com/BrunoCarvalhoFeitosa/layers-saas.git
   ```
2. Instale os pacotes do NPM
   ```sh
   npm install ou yarn
   ```
   
3. Inicie o projeto
   ```sh
   npm run dev ou yarn dev
   ```
